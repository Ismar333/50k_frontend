1) Правила подключения стилей прямо на странице
    используется тег  style
    
    <style>
        Правила стилей
    </style>

    - Данный кусок кода CSS должен распологаться перед закрывающим тегом </head>

2) Правила написания стилей
    - Начинается все с того, кому мы должны присвоить стили?
        class
        id
        tag
        attribute
    - мы должны открыть и закрыть фигурную скобку
        {  }

        Целостная картина равна:
        class {

        }
        
        id {

        }

        tag {

        }

        attribute {

        }
3) Закрывающая фигурная скобка должна быть паралельна названию класса или же любого свойства.
