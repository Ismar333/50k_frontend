Правила HTML
	0) После того как мы создадим тег Нажать на ENTER
	1) Правила табов
		- прежде чем писать код либо содержимое нужно убедится что в начале открытого и закрытого тега есть 1 TAB
	2) Каждый код должен находится между открытым и закрытым тегом
	3) Каждый тег должен начинаться с < и заканчиваться >
	4) Каждый открытый тег должен быть параллен закрытому. И наоборот, каждый закрытый тег должен быть параллен открытом
		Например: 
		<div>
		|	Содержимое
		|	Содержимое
		|	Содержимое
		|	Содержимое
		</div>
	5) ВСЕГДА НАБЛЮДАЙТЕ ЗА АКТИВНОЙ ПОЛОСОЙ КОТОРАЯ НАХОДИТСЯ СЛЕВА ОТ ТЕГА - НА НЕЙ НИЧЕГО НЕ ДОЛЖНО БЫТЬ
		<div>
		|	Содержимое
		|	Содержимое
		|	Содержимое
		|	Содержимое
		</div>

	6) Каждый ВНУТРЕННИЙ тег либо содержимое должна создержить 1 ТАБ в право
		<div>
		|ТАБ Содержимое
		</div>
	7) Каждый СОСЕДНИЙ ТЕГ не должен содержить дополнительный таб в право
		Блок 1
		<div>
		|ТАБ Содержимое
		</div>

		Блок 2
		<div>
		|ТАБ Содержимое
		</div>

	8) Стрелочка скрытия кода
		Всегда пробуйте свернуть свой код.
		Если не получается его свернуть - ЗНАЧИТ СТРУКТУРА НЕ ПРАВИЛЬНАЯ



Правило атрибутов

	1) class="" - Каждый атрибут должен начинатся со СВОЙСТВА! CLASS / ID / DATA-USER
				- Далее пишется равно = без каких либо пробелов!
				- Ковычки после равно пишутся, без пробелов!
				- Если Ковычка начинается с двойной " она обязательно должна заканчиваться двойной "
				- Если Ковычка начинается с одинарной ' она обязательно должна заканчиваться одинарной '
